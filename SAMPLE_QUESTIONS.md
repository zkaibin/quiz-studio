- Progressive difficulty from easy to hard

## üß™ How Questions Are Generated

When a quiz is generated, the system:

1. **Selects a template** based on category and difficulty
2. **Substitutes character names** randomly from the pool
3. **Generates random numbers** (1-10) for {num1}, {num2}, etc.
4. **Creates the full question** with personalized content
5. **Provides the answer** calculated from template

### Example Generation

**Template:**
```
{character1} has {num1} apples. {character2} has {num2} apples. 
How many apples do they have together?
```

**Generated Question 1:**
```
Elsa has 7 apples. Olaf has 4 apples. 
How many apples do they have together?
Answer: 11
```

**Generated Question 2:**
```
Mickey Mouse has 2 apples. Elsa has 9 apples. 
How many apples do they have together?
Answer: 11
```

*Note: Each time a quiz is generated, the random numbers and character names change!*

## üìù Adding More Questions

You can easily add more questions through the admin panel:

1. Open **Admin Panel**: http://localhost:3000/admin
2. Go to **Questions** tab
3. Click **Add Question**
4. Fill in:
   - Category (addition, subtraction, etc.)
   - Difficulty (easy, medium, hard)
   - Template (with {placeholders})
   - Placeholders (list like: ["character1", "num1", "num2"])
   - Answer (calculated value)
5. Click Save

The new question will immediately be available for quizzes!
